# Sphoorthy-Treasure-Hunt
A gamified online treasure hunt inspired by Sphoorthy's wisdom, humor, and challenges, featuring interactive riddles, stories, and level-based gameplay.

## Running the APP

`cd Treasure_Hunt`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`

### Backend
1. `cd server`
2. `npm install`
3. `npm start`
